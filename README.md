## `> cat modules/`

<details>
<summary><b>рҹ”— Subdomain Finder</b> вҖ” crt.sh certificate transparency</summary>

Queries SSL certificate transparency logs вҖ” every subdomain that's ever had a cert issued shows up here. No API key. No rate limits published. Returns forgotten dev servers, admin panels, staging environments, and internal microservices.

```
crt.sh/?q=%.target.com&output=json
вҶ’ admin.target.com
вҶ’ staging-api.target.com
вҶ’ old.target.com
вҶ’ internal-dashboard.target.com
вҶ’ ... 39 more
```

</details>

<details>
<summary><b>рҹ“Қ IP Intelligence</b> вҖ” ip-api.com</summary>

One free API call returns: city, region, country, ISP, organization, ASN, timezone, and detection flags for Tor exit nodes, VPN providers, datacenter hosting, and open proxies. 45 requests/minute free, no key.

```json
{
  "city": "Frankfurt",
  "isp": "Hetzner Online GmbH",
  "org": "AS24940 Hetzner Online GmbH",
  "proxy": false,
  "hosting": true,
  "tor": false
}
```

</details>

<details>
<summary><b>рҹ“Ӣ DNS Record Viewer</b> вҖ” Google DNS-over-HTTPS</summary>

Parallel queries for A, AAAA, MX, TXT, NS, CNAME records using Google's DoH JSON API. Free, no key. Reveals email provider (MX), CDN (A records pointing to Cloudflare/Fastly), and misconfigurations (SPF/DMARC failures).

```
A     вҶ’ 104.21.33.167 (Cloudflare)
MX    вҶ’ mail.google.com (Google Workspace)
TXT   вҶ’ v=spf1 include:_spf.google.com ~all
DMARC вҶ’ v=DMARC1; p=reject; rua=mailto:dmarc@target.com
```

</details>

<details>
<summary><b>рҹ•°пёҸ Wayback Machine</b> вҖ” Internet Archive CDX API</summary>

Free public API. Returns first archived date, last snapshot, total snapshot count, and recent URLs. Reveals when a domain first appeared, deleted pages, and historical content вҖ” including content removed after incidents.

```
First seen:    2011-03-15 (14 years ago)
Last snapshot: 2026-05-01
Total snaps:   2,847
Status codes:  200 (2,401) В· 301 (312) В· 404 (134)
```

</details>

<details>
<summary><b>рҹ‘Ө Username Enumeration</b> вҖ” HTTP status probing</summary>

Fires parallel HEAD requests to 50+ known profile URL patterns. No API key needed вҖ” just reads HTTP status codes. 200 = profile exists, 404 = doesn't. Covers: GitHub, Reddit, Twitter, Instagram, TikTok, Pinterest, Steam, Spotify, Twitch, YouTube, Keybase, Dev.to, Medium, npm, PyPI, HackerNews, GitLab, Telegram, Snapchat, and more.

</details>

<details>
<summary><b>рҹҗҷ GitHub Deep Recon</b> вҖ” GitHub REST API</summary>

Beyond basic profile/repo lookup: extracts real email addresses from public commit metadata вҖ” even when hidden in profile settings. Also surfaces language breakdown, contribution frequency, linked social accounts, and starred repo topics to infer interests and expertise.

</details>

<details>
<summary><b>рҹ”ҙ Reddit User Analysis</b> вҖ” Reddit JSON API</summary>

No key required. Returns: account age, link/comment karma split, top subreddits by post count, posting frequency by hour (reveals timezone), trophy list, and most recent 25 post titles. The posting hour histogram alone often reveals a target's real timezone.

</details>

<details>
<summary><b>рҹ”„ Reverse IP Lookup</b> вҖ” HackerTarget</summary>

Finds all domains hosted on the same IP address. 100 requests/day free, no key. Reveals domain clusters вҖ” fake news networks, spam farms, related shell companies, or all projects by the same solo developer on shared hosting.

</details>

<details>
<summary><b>рҹ”Ң Port Scanner</b> вҖ” HackerTarget nmap wrapper</summary>

Scans the top 20 common ports on any domain or IP via HackerTarget's free nmap API. Returns open/closed/filtered status with service names: SSH, HTTP, HTTPS, FTP, SMTP, MySQL, PostgreSQL, RDP, MongoDB, Elasticsearch. 100 req/day free.

</details>
